variable : 

var -> is keyword in js that can be redeclared and updated. It is functional scoped.

let -> is keyword in js that cannot be redeclared but can be updated. it is block scoped.

const -> is keyword in js that cannot be redeclared or updated. it is block scoped.

Hoisting : 

Hoisting means js move declarations to top.

for var it gives undefined 
for const and let it gives temporal dead zone
